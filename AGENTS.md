# Default Vite Sandbox

- Project root: `/workspace/template`
- Preview URL: Vite dev server on port `5173`
- Package manager: `pnpm`

## Common commands

- `pnpm dev` to run the app manually
- `pnpm build` to verify the template builds
- `pm2 status` to inspect the background dev server

## Working rules

- Keep project files inside `/workspace/template`
- Do not edit generated build output in `dist/`
- Use the non-root `claude` user for shell work
