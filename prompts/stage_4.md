#### 2.3 Interview Best Practices

**Opening Approach:**
- Begin with open-ended questions to let management tell their story
- Build rapport before diving into challenging questions
- Demonstrate that you've done your homework—reference specific data points

**Questioning Techniques:**
- Start broad, then narrow: "Tell me about your customer base" → "Why did Customer X decline 20%?"
- Use the "funnel" approach: open → probing → closed → confirmatory
- When investigating sensitive areas, use the "we noticed" framing: "We noticed a $2M variance in..."
- Always ask "what else should we know about [topic]?" before moving on

**Documentation Protocol:**
- Assign a dedicated note-taker for each session
- Record management representations verbatim where possible
- Note who made each representation (name and title)
- Flag any inconsistencies with written data immediately
- Document items for follow-up in real-time
